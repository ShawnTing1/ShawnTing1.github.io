丁硕的个人博客

这是丁硕的个人博客的源代码和文件。

index.html - 个人博客的主页文件，包含了整个网站的结构和内容。
style.css - 个人博客的样式文件，用于自定义网站的外观和布局。
images/ - 存放用于网站的图片文件夹。
blog/ - 存放丁硕的博文文件夹。
    ├── 2024/
    │   ├── 01/
    │   │   ├── article1.html
    │   │   └── article2.html
    │   └── 02/

    ├── 2023/
    └── ...

如何使用：

1. 将所有文件和文件夹保留在同一个目录中。
2. 将index.html文件放在你的网站根目录下。
3. 可以根据需要修改index.html文件中的内容，包括标题、导航链接和博文内容。
4. 如果需要自定义样式，可以在style.css文件中进行修改。
5. 将图片文件放置在images/文件夹中，并在index.html中引用它们。
6. 如果需要添加新的博文，请按照blog/目录中的结构创建新的HTML文件，并在index.html中添加链接。

注意事项：

- 请确保在将文件上传到网站服务器之前进行适当的测试，以确保所有链接和资源文件的路径正确。
